计算机网络之域

一、域的概述

  1. 说明：
    • 域（Domain）是计算机网络的一种形式，其中所有用户账户、计算机、打印机和其他安全主体，都在位于称为域控制器的一个或多个中央计算机集群上的中央数据库中注册
    • 域之间相互访问则需要建立信任关系；信任关系是连接域与域之间的桥梁。当一个域与其他域建立了信任关系后，2个域之间不但可以按需要相互进行管理，还可以跨网分配文件和打印机等设备资源，使不同的域之间实现网络资源的共享与管理，以及相互通信和数据传输
    • 域既是 Windows 网络操作系统的逻辑组织单元，也是Internet的逻辑组织单元，在 Windows 网络操作系统中，域是安全边界
    • 域管理员只能管理域的内部，除非其他的域赋予他管理权限，他才能够访问或者管理其他的域，每个域都有自己的安全策略，以及它与其他域的安全信任关系
  2. 特点：集中/统一管理
  3. 域的组成：域控制器：DC（Domain Controller）、成员机
  4. 域的部署：
    • 安装了活动目录，就生成了域控制器；活动目录：AD （Active Directory）
    • 有了域控制器，就生成了域环境

二、内网环境

  1. 工作组：属于计算机的默认模式，人人平等，不方便管理
  2. 域：人人不平等，集中管理，统一管理

三、安装活动目录（AD）

  1. 打开win2008服务器并指定网卡、配置静态IP

  2. 运行win+ R，输入dcpromo，开始安装活动目录，进入安装向导

  3. 选择安装与域服务器和DNS服务器（必须选择）

  4. 创建一个域

  5. 自定义域名

  6. 下一步到如下图

  7. 设置域服务还原密码（此密码并非域密码）

  8. 检查域服务器的配置，无错误就下一步，之后等待配置，完成后重启服务器即可

  9. 登陆域服务器，重启系统后，之前的不同服务器就变成了域服务器，密码还是之前的服务器密码

  10. 检查服务器是否成功转换为域服务器

  12. 打开域用户域计算机

四、添加域成员

添加winXP成员机

  1. 打开winXP系统，设置与域服务器在同一网卡、配置同网段IP、DNS服务器（客户机与服务器互通）

  2. 给此成员机添加域

  3. 以域管理员身份登陆，添加成功后，重启此系统，此系统已成功加入test.com域，要想登陆此计算机，必须在域服务器上创建账户并授权

添加win7成员机

  1. 打开win7系统，设置与域服务器在同一网卡、配置同网段IP、DNS服务器（客户机与服务器互通）

  2. 给此成员机添加域

  3. 以域管理员身份登陆，添加成功后，重启此系统，此系统已成功加入test.com域，要想登陆此计算机，必须在域服务器上创建账户并授权

  4. 打开域服务器，查看计算机是否添加成功

七、域中的组织单位（OU）

  1. 说明：组织单位(Organizational Unit)简称OU 是可以将用户、组、计算机以及其它组织单位，放入其中的AD(Active Directory，活动目录)容器，是可以指派组策略设置或委派管理权限的最小作用域或单元

  2. 打开活动目录，创建OU

  3. 自定义OU名称

  4. 按照公司的组织架构创建OU，示例如下（注意图标变化）

  5. 移动域用户到指定的组织单位（注意：域用户一般不要删除，就算新建一个相同用户，也回不到删除之前了）

  6. 查看是否移动成功

八、域中的组策略（GPO）

  1. 作用：通过组策略（Group Policy Object）可以修改计算机的各种属性，如开始菜单、桌面背景、网络参数等
  2. 组策略在域中，是基于OU来下发的
  3. 组策略在域中下发后，用户的应用顺序是：LSDOU（L：本地；S：站点；D：域；OU：组织单元）

  2. 给每个部门都创建一个组策略

  3. 自定义组策略表名称

  4. 给所有分支都添加GPO

十、组策略执行优先级

LSDOU顺序

  1. 设置IT部的域用户为：mazi.wang

  2. 设置东北大区的域用户为：san.zhang

  3. 在乐天集团组策略表中，启用删除DOS窗口策略

  4. 在IT部组策略表中，禁用删除DOS窗口策略

  5. 打开成员机win7，登陆san.zhang域账户，发现DOS窗口无法运行，这是因为东北大区没有自己配置对应的策略，只能继承乐天集团的组策略

  6. 切换账户，登陆mazi.wang域账户登陆成员机，发现DOS窗口成功运行，这是因为IT部自己配置了对应的策略，就不用继承乐天集团的组策略了（前提条件：上级组策略未开启强制设置）

阻止继承

  1. 打开组策略，给指定组织单位设置阻止继承，此时的组织单位只执行自己的组策略，上级的组策略失效，不继承

强制执行

  1. 打开域服务器的组策略管理，选择乐天集团的组策略，右键设置强制，此时的IT部自己的组策略全部失效，其他的下级策略也全部失效，只能强制继承乐天集团的组策略，就算组织单位选择了阻止继承，也没用

十二、域账户完全控制权限

  1. 说明：域账户的完全控制权限，仅限于对成员机（计算机）生效

  2. 用域管理员账户test.com\administrator登陆成员机win7，打开计算机管理，将域账户mazi.wang添加到本地管理员组

  3. 用域账户mazi.wang登陆成员机win7，以后在此计算机上安装、卸载程序等一系列管理员操作，都可以直接操作，无需请求域管理员同意
