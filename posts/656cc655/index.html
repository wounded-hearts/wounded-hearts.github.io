一、防火墙的基本概念

  1. 防火墙的定义：是一款具备安全防护功能的网络设备
  2. 作用：隔离网络（将需要保护的网络与不可信任的网络进行隔离，隐藏信息并进行安全防护）
  3. 防火墙基本功能：
    • 访问控制
    • 攻击防护
    • 冗余设计（rong）
    • 路由、交换
    • 日志记录
    • 虚拟专网VPN
    • NAT
  4. 防护墙与路由器的最大区别
    • 路由器只能双向访问
    • 防火墙可以单向访问

二、防火墙区域隔离

  1. 防火墙区域分类

    • 内部区域：存放机密数据，禁止DMZ区域向内部区域写任何策略，可在此线路上添加IPS（入侵防御系统）

    • DMZ区域：称为隔离区，也称非军事化区 / 停火区，可在此线路上添加IDS（入侵检测系统）

    • 外部区域：即为外网（互联网）

  2. 什么时候需要隔离区：当内网向外网发布服务器的时候

三、防火墙的分类

  1. 按防火墙形态：

    • 软件防火墙
    • 硬件防火墙

  2. 按技术实现：

    • 包过滤防火墙

    • 状态检测包过滤防火墙

    • 应用（代理）防火墙

    • WAF防火墙

    • 应用层防火墙

  3. 防火墙的发展历史

    • 包过滤防火墙：最早的防火墙技术之一，功能简单，配置复杂（已淘汰）
    • 应用网关/应用代理防火墙：最早的防火墙技术之二，连接效率低，速度慢（已淘汰）
    • 状态检测防火墙：现代主流防火墙，速度快，配置方便，功能较多
    • DPI防火墙：全称Deep Packet Inspection未来防火墙的发展方向，能够高速的对第七层数据进行检测

  4. 状态检测防火墙处理示例图

  5. 衡量防火墙性能的5大指标：

    • 吞吐量：在不丢包的情况下单位时间内通过的数据包的数量
    • 时延：数据包第一个比特进入防火墙到最后一比特从防火墙输出的时间间隔
    • 丢包率：通过防火墙传送时所丢失数据包数量占所发送数据包的比率
    • 并发连接数：防火墙能够同时处理的点对点连接的最大数目
    • 新建连接数：在不丢包的情况下每秒可以建立的最大连接数

四、防火墙的工作模式

透明模式

  1. 示例图：

  2. 说明：透明模式/桥模式一般用于用户网络已经建设完毕，网络功能基本已经实现的情况下，用户需要加装防火墙以实现安全区域隔离的要求（不改变网络结构的情况下）

  3. 作用：充当二层交换机，只是多了网络数据过滤功能

路由模式

  1. 示例图：

  2. 说明：

    • 路由/NAT模式一般用于防火墙当作路由器和NAT设备连接上网的同时，提供安全过滤功能
    • SNAT：源转换
    • DNAT：目标转换

  3. 作用：充当三层交换机，并提供网络数据过滤

混杂模式

  1. 示例图：

  2. 说明：充当三层交换机，做路由与交换功能，并提供网络数据过滤

  3. 作用：一般网络情况为透明模式和路由模式的混合。

创建ACL

  • 作用：决定源IP与目的IP的数据是否可以互通

配置SNAT

  • 作用：使得私网IP可以访问公网IP

配置DNAT

  1. 作用：使得外界公网可以访问内部私网数据（例如：将DMZ区域的web网站发布到公网）

  2. 添加web服务器的公网地址，可以让其他公网访问

  3. 添加web服务器的私网IP，也是公司web服务器的真实IP

  4. 配置DNAT

  5. 添加ACL（添加放行策略，若不添加，防火墙会将数据拦截，使其无法访问web服务器）

六、配置备用防火墙

HA概述

  1. 高可用性(HA)系统
  2. 两个防火墙之间连接的是心跳线，一般用光纤，这根线可以检测对方状态并可以同步数据；这就保证了防火墙的HA高可用性
  3. 作用：当主防火墙发生故障时，备用防火墙会立刻启动，继续保护网络安全
