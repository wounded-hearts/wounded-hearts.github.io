kali系统的安装与配置
发表于 2021-05-19| 更新于 2021-05-19|工具
字数总计:594|阅读时长: 2 分钟
|阅读量:

一、下载镜像文件

  • 官网下载：点击进入

  • 推荐版本下载：

二、创建kali虚拟机

  1. 新建虚拟机 > 典型 > 稍后安装操作系统

  2. 选择对应的版本

  3. 设置安装位置与名称

  4. 设置磁盘大小（推荐100GB，但是实际不会占用100GB）

  5. 设置虚拟机硬件

三、安装kali系统

  1. 启动kali，选择图形界面安装

  2. 选择中文简体语言

  3. 选择区域：中国

  4. 配置键盘：汉语

  5. 自动获取IP地址

    • 获取失败

    • 解决方案：重新设置网卡，然后继续，重新获取（注意应用、确认）

  6. 自定义主机名：wlet.cn

  7. 自定义用户名：wlet

  8. 自定义用户密码：123456

  9. 磁盘分区

  10. 选择工具安装包

  11. 安装GRUB启动引导器

  12. 安装完成后，继续，自动进入登陆页面，输入用户名、密码即可进入kali系统（无须安装VMware Tools，新版本已默认安装）

四、设置root用户

  1. 打开kali终端：ctrl + alt + T

  2. 设置root用户的密码：sudo passwd root

  3. 暂时切换到root用户：su root

  4. 切换到真正的root用户：su - root

  5. 查看当前系统使用的shell类型：echo $SHELL（注意大写）

    • bash：centos系统所使用的shell
    • zsh：kali系统所使用的shell，兼容bash的用法和命令

五、更换apt源并更新软件

  1. 切换root用户：su - root

  2. 打开配置文件：vim /etc/apt/sources.list （末行模式:set nu显示行号）

  3. 注释掉自带的源地址，并添加国内的源地址（中科大kali源）

    • deb：代表软件的位置
    • deb-src：代表软件的源代码的位置

  4. apt基本命令

    • apt update：从/etc/apt/sources.list文件中定义的源中获取的最新的软件包列表（不更新，只检查）
    • apt upgrade（推荐）：如果软件包有相依性的问题，此软件包就不会被升级
    • apt dist-upgrade（不推荐）：升级时，如果软件包有相依性问题，会移除旧版，直接安装新版本
    • apt install：安装软件包
    • apt remove：移除软件包

  5. 升级软件包：先apt update再apt upgrader
