Python3常用模块的用法(十六)

random模块

  1. 作用：产生一个随机数

  2. 导入模块：import random

  3. random模块中的内建函数：

    • random.random()：产生(0,1)之间的随机数，float型

    • random.randint(a, b)：产生[a,b]之间的随机整数，int型

    • random.randrange(a, b, c)：产生[a,b,c)，产生a~b之间，步长为c（step=c）的随机整数，int型

    • random.choice(seq)：随机选择序列中的一个元素，seq代表可迭代的序列

    • random.choices(seq)：随机选择序列中的一个元素，并将此元素放入list中

      • k代表选取的次数，weights代表每个元素的选择概率（必将元素一一对应）

      • 示例：

    • random.shuffle(list)：将列表中的元素顺序打乱（仅限于list）

  4. 应用：验证码

os模块

  1. 作用：python中的操作模块，可以处理文件和目录（文件夹）

  2. 导入模块：import os

  3. 注意事项：path代表绝对路径和相对路径

  4. os模块下的内建函数

    • os.getpid()：获取自身的进程号
    • os.getppid()：获取父进程的进程号
    • os.getcwd()：获取当前文件夹的绝对路径（无参）
    • os.listdir(path)：遍历文件夹，获取指定路径下的所有文件，并保存在list中（无参默认路径为当前）
    • os.remove(path)：删除指定的文件（必须要写路径）
    • os.mkdir(path)：创建一个文件夹（必须要写路径）
    • os.rmdir(path)：删除空文件夹（必须要写路径，只能删除空的）
    • os.chdir(path)：切换文件夹（切换后，用os.getcwd()查看是否切换成功）

  5. os.path中的内建函数

    • os.path.dirname(path)：获取文件夹的路径（参数为__file__代表当前文件夹的路径）
    • os.path.abspath(path)：获取文件或文件夹的路径（参数为__file__代表当前文件的路径）
    • os.path.split(path)：将获得的文件或文件夹路径一分为二，并将其放入tuple中；默认分隔符为最后一个//或\\
    • os.path.splitext(path)：将获得的文件路径一分为二，并将其放入tuple中；默认分隔符为最后一个.（必须是文件的路径)
    • os.path.join(path, file_name)：拼接文件的路径（字符串的拼接）
    • os.path.getsize(path)：获取文件或文件夹的大小
    • os.path.exists(path)：判断文件或文件夹是否存在
    • os.path.isabs(path)：判断path是否为绝对路径格式，不在乎路径是否存在
    • os.path.isdir(path)：判断路径是否存在

  6. 应用：文件复制

sys模块

  1. 作用：提供对解释器使用或维护的一些变量的访问，以及与解释器强烈交互的函数
  2. 导入模块：import sys
  3. sys模块中的内建函数
    • sys.exit(n)：退出程序，正常退出时，n=0
  4. sys模块中的内建参数
    • sys.version：获取python解释程序的版本信息
    • sys.path：返回模块的搜索路径，初始化时使用python环境变量的值

time模块

  1. 作用：获取时间和修改时间
  2. 导入模块：import time
  3. time模块中的内建函数：
    • time.time()：获取时间戳（以秒为单位）
    • time.sleep(n)：时间延时，当n=2时，延时2秒
    • time.ctime()：将时间戳转换为str型；输出（星期，月，日，时分秒，年）
    • time.localtime()：将时间戳转换为tuple型
    • time.mktime()：将tuple型时间转换为时间戳，此时的时间戳保留一位小数（四舍五入）
    • time.strftime()：将tuple型的时间，转换为自定义str型时间；此时的tuple型由底层完成，无需手动
      • 示例：time.strftime('%Y-%m-%d')
    • time.strptime()：将str型时间，转换为自定义tuple型时间
      • 示例：time.strptime('2020/05/22','%Y/%m/%d')（年月日必须一一对应，并且年份必须大写）

datetime模块

  1. 作用：获取时间和修改时间
  2. 导入模块：import datetime
  3. datatime模块中的内建函数：
    • datetime.time()：输出初始化如期：00:00:00
    • datetime.date(*args)：自定义日期
    • datetime.datetime.now()：获取当前时间
    • datetime.timedelta(hours=n)：设置时间差（hours可以替换为其他的关键字，如days等）
      • 示例一：datetime.timedelta(hours=2) + datetime.datetime.now() （利用时间差，重定义时间，延时两个小时）
      • 示例二：datetime.datetime.now() + datetime.timedelta(hours=-2)（提前两个小时）
