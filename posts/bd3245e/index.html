CentOS8的安装与使用
发表于 2020-09-07| 更新于 2020-09-07|Linux
字数总计:411|阅读时长: 1 分钟
|阅读量:

阿里镜像目录aarch64、ppc641e、x86 64的区别

  1. x86 64：主流的64位架构，代表厂家有Intel、AMD
  2. aarch64：是arm架构的64位系统，基于armv8指令集的64位架构，应该是后续ARM主流商用服务器的标准
  3. ppc641e：纯小端模式，POWER8作为首要目标，OpenPower基金会基础的技术，试图使基于x86的Linux软件的移植工作以最小的工作量进行

CentOS安装

  1. 参考文献：点击进入
  2. 注意事项：镜像文件与安装源，根据自己需要一一对应安装（目录选择）
  3. 阿里镜像：点击进入（选择boot.iso文件即可）
  4. 安装源：点击进入

图形化界面（桌面）安装

  1. 参考文件：点击进入
  2. 使用root用户登录
  3. 尝试启动图形化界面：startx（启动失败，进入一下步骤）
  4. 下载软件：yum grouplist
  5. 安装桌面（图形化界面）：yum groupinstall "Server with GUI" -y 耐心等待即可（-y表示自动选择确认）
  6. 安装完成后启动：startx或init 5 （每次开机都要重新启动）

临时修改系统语言（默认英文）

  1. 参考文献：点击进入

  2. 查看当前语言类型：locale 或者 echo $LANG

  3. 查看本地语言字符集：localectl

  4. 修改语言类型：LANG="zh_CN.UTF-8" （每次启动桌面前，都要重新设置）

  5. 启动图形化界面：startx

CentOS8安装中文输入法

  1. 安装输入法：yum install ibus-libpinyin.x86_64 -y：或：yum install ibus ibus-libpinyin -y

  2. 设置：汉语（智能拼音）

  3. 重启系统：init 6

  4. 注意：先查看语言类型，再启动图形化界面
