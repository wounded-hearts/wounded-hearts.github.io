多级标题

  • 快捷键：ctrl + 1~6 （不要使用单独的数字按键，最多只能写到六级标题）
  • 输入法：
    1. # 一级标题 (注意#号后面有一个空格，以下同理)
    2. ## 二级标题
    3. ### 三级标题
    4. #### 四级标题
    5. ##### 五级标题
    6. ###### 六级标题

下划线

  • 演示：下划线
  • 快捷键：ctrl + U
  • 输入法：<u>下划线</u>

字体加粗

  • 演示：字体加粗
  • 快捷键：ctrl + B
  • 输入法：**字体加粗**

字体倾斜

  • 演示：字体倾斜
  • 快捷键：ctrl + i
  • 输入法：*字体倾斜*

删除线

  • 演示：删除线
  • 快捷键：shift + alt + 5 （使用%5这个按键）
  • 输入法：~~删除线~~

上下标

  • 开启上下标功能：文件 – 偏好设置 – Markdown – Markdown扩展语法（勾选上下标等信息，注意重启）

  • 上标：

    1. 演示：a^2^ + b^2^ = c^2^
    2. 输入法：a^2^ + b^2^ = c^2^

  • 下标：

    1. 演示：H2O
    2. 输入法：H~2~O

有序列表

  • 演示（每行前面都有序号）：
    1. 有序列表，第一行
    2. 有序列表，第二行
    3. 有序列表，第三行
    4. 有序列表，。。。
  • 快捷键：ctrl + shift + [
  • 输入法：1.空格 100.空格（编号后一定要加空格，行末enter键，序号会在当前序号添加并+1；行末上下键翻动，序号不会添加）
  • 降级快捷键：tab
  • 升级快捷键：shift + tab

无序列表

  • 演示：
    • 无序列表，第一行
    • 无序列表，第二行
    • 无序列表，第三行
    • 无序列表，。。。
  • 快捷键：ctrl + shift + ]
  • 输入法：*空格或-空格（符号后一定要加空格，行末enter键，列表会自动添加一行；行末上下键翻动，列表不会添加）
  • 降级快捷键：tab
  • 升级快捷键：shift + tab

注意：有序列表和无序列表可交错使用（有序中有无序，无序中有有序）

任务列表

  • 演示（可以选择任务）：
    • 任务列表，第一行
    • 任务列表，第二行
    • 任务列表，第三行
    • 任务列表，。。。
  • 快捷键：无
  • 输入法：- [ ]（每一个符号后都要加一个空格隔开，行末enter键，列表会自动添加一行；行末上下键翻动，列表不会添加）
  • 降级快捷键：tab
  • 升级快捷键：shift + tab

注意：任务列表也可以和有序列表和无序列表一起交错使用 （如上面演示）

表格插入

  • 演示：

    学号 姓名 年龄
    1001 张三 66
    1002 李四 88
    1003 王五 99

  • 快捷键：ctrl + T

  • 输入法：|学号|姓名|年龄|（回车即可创建表格）

  • 添加下一行：ctrl+ 回车键

  • 调整表格：点击创建的表格，会有很多按钮，自己去尝试即可
