一、VPN概述

  1. 全称：Virtual Private Network （虚拟专有网络、虚拟专网）
  2. 作用：
    • VPN可以实现在不安全的网络上，安全的传输数据，类似专网
    • VPN只是一个技术，使用PKI技术，来保证数据的安全三要素
  3. 安全三要素：完整性、机密性、身份验证
  4. 路由器的工作原理：内网 >> 外网（顺序）：路由 >> NAT >> VPN >> 出去
  5. VPN隧道的优点：
    • 安全性
    • 合并两家企业内网
  6. VPN隧道技术：重新封装技术+加密认证技术
    • 传输模式：只加密上层数据，不加密私有IP包头，速度快
    • 隧道模式（默认推荐）：加密整个私有IP包，包括IP包头，更安全，速度相对慢

二、VPN的类型

远程访问VPN

  1. 条件:
    • 在公司需要搭建VPN服务器
    • VPN服务器需要对VPN客户端进行身份验证
    • VPN服务器需要给VPN客户端下发权限及IP地址
  2. 作用：
    • 一般用在个人到安全连接企业内部
    • 一般出差员工/在家办公，安全连接内网时使用
    • 一般公司部署VPN服务器，员工在外拨号连接VPN即可
  3. 常见RA-VPN协议：PPTP VPN、L2TP VPN、SSTP VPN、EZ VPN（easyvpn）、SSL VPN

点到点VPN

  1. 一般用在企业对企业安全连接
  2. 一般需要在两个企业总出口设备之间建立VPN通道
  3. 常见的点到点VPN：IPsecVPN （国际通用）
  4. IPsecVPN：属于点到点VPN，可以在2家企业之间建立VPN隧道

创建扩展ACL

  1. 进入路由器一的配置终端

  2. 定义内网接口

  3. 定义外网接口

  4. 查看已有的acl表：show ip acce （防止覆盖已有的acl表）

  5. 定义acl表：

    • acc 101 deny ip 192.168.1.0 0.0.0.255 172.16.1.0 0.0.0.255 （禁止192访问172）
    • acc 101 permit ip any any （允许任意IP访问任意IP）

PAT动态映射

  • 做PAT动态映射：ip nat inside source list 101 int f0/0 overload
  • 作用：给内部地址池做动态映射，并指定出接口f0/0，并且开启端口复用技术，实现私网地址访问互联网

八、远程访问VPN实验

实验拓扑图

  • 实验目的：实现100.1.1.2访问200.1.1.2

配置国外服务器

  1. 打开国外服务器，设置虚拟网卡为VMnet2

  2. 进入服务器，配置静态IP为200.1.1.2

  3. 在此服务器上发布一个web网站

配置VPN服务器

  1. 打开win2003服务器（VPN服务器），额外添加一个虚拟网卡，并将两个虚拟网卡分别设置为VMnet1和VMnet2

  2. 进入服务器，给VMnet1设置静态IP为100.1.1.1；VMnet2设置静态IP为200.1.1.1

  3. 判断哪个网卡是VMnet1，需要打开虚拟机设置，关闭其中一个网卡，即可判断

  4. 当国外服务器配置完成后，确认能ping通国外服务器，并且可以访问到web网站

  5. 进入VPN配置界面

  6. 选择自定义配置

  7. 勾选相关服务，下一步确认安装即可

  8. 查看VPN服务是否开启，PPTP VPN、L2TP VPN的默认端口号：TCP 1723

  9. 设置网络地址转换池（NAT）

  10. 选择接口类型

  11. VPN服务器创建一个普通账户，并开启远程访问权限

  12. 设置VPN服务器的IP地址池

配置win7系统

  1. 打开win7系统虚拟机，设置虚拟网卡为VMnet1

  2. 进入系统，设置静态IP为：100.1.1.2

  3. 开始安装VPN客户端，选择连接到工作区

  4. 选择VPN连接

  5. 选择稍后连接

  6. 设置VPN服务器的IP，并自定义名称

  7. 输入VPN账户密码即可

  8. 进入网络连接，即可查看VPN客户端，点击进入，可更换VPN账户密码

  9. 访问国外服务器上的web网站
