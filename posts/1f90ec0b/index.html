Web Scraper使用教程

视频教程

  • 百度云盘：
    1. 链接：点击进入
    2. 提取码：59c1

通用爬取步骤

启动爬虫插件

  • 示例网站：http://www.0797rs.com/

  • 打开Google浏览器，键盘按F12或右键选择检查，再点击web scraper，进入插件

创建项目

  1. 第一步：依次点击Create new sitemap >> Create Sitemap

  2. 第二步：填写项目名称（任意名称）、目标地址（爬取数据的URL）

  3. 第三步：填写完成后点击Create Sitemap，即可创建项目根目录_root

  4. 第四步：点击Sitemaps查看所有的项目

创建选区

  1. 进入name01项目，点击Add new selector，进入选区设置界面

  2. id可以为任意三个以上字符名称；type属性为Link（可爬去链接）

  3. Multiple复选框，获取多个数据（一般都需要选择）

  4. 重要操作：点击select后，继续点击网页中需要的数据（相同类型的数据，点击2到3个即可自动选择剩余的相同类型数据），当需要的所有数据都被选中后，点击Done selsecting，最后Save select即可

  5. 查看获取到的数据：点击Data preview即可查看获取到的数据，若数据只有一条或没有，点击Edit重新设置选区，选择Multiple

数据爬取

  1. 依次点击Sitemap name01 >> Scrape >> start scraping >> refresh

    • name01为项目名称；
    • 时间一般为默认；
    • 爬取需要时间，需等待

  2. 点击refresh后，查看大概的数据是否正确、完整

数据保存

  • 依次点击：Sitemap name01 >> Export data as CSV >> Download now!

常用选择器类型

  • 选择器类型是在创建选区时设置的（非常重要的设置）

文本类型

  • 进入选区，选择Type的属性值为Text即可

链接类型

  • 进入选区，选择Type的属性值为Link即可

图表类型

  1. 进入选区，选择Type的属性值为Table

  2. Header row selectot与Data rows selector的选区可根据自己需要，在指定的选区再次选择一个小的选区

  3. Table columns的相关列属性，可根据自己需要选择

  4. 此插件中的所有输入框中的数据，必须大于三个字符

同一页面爬取多种数据内容

  1. 注意事项：

    • 必须先创建一个大选区，并且大选区的Type属性值为Element
    • 大选区可自定义勾选Multiple，若大选区有多个，必须选择Multiple，若大选取只有一个可不选择

  2. 新建项目 >> 创建大选区（Type的属性值必须为Element；id为任意大于等于三个字符）

  3. 进入大选区内部 >> 创建多个小选区分支

  4. 小选区设置（必须在网页上标注的选区中创建小选区，一般为网页第一条个数据，会有一个黄色框显示）

  5. 查看获取到的数据：返回项目根目录_root，在大选区all上点击Data preview

  6. 先运行数据爬取，再选择数据保存

元素选择器

滚动加载

  1. 说明：在网站中，每页的数据量很大，所以刚开始只显示部分数据，若要显示全部数据，就要向下滚动，一边滚动一边加载，一直到底部才能加载完整个页面的数据
  2. 示例网站：https://www.jd.com/
  3. 注意事项：
    • 若将Type的属性值设置为Element，只能获取到的部分数据，未加载的数据无法获取
    • 将Type的属性值设置为Element scorll down，可一边加载，一边获取，最终获取整个页面的全部数据
    • 其他操作和同一页面爬取多种数据内容基本一致

参数翻页

  1. 说明：观察网站的每页数据，找到有规律的数值，将可变数值替换为特定的表示法
  2. 重要操作：在创建项目时，只需将URL改为可变化的特殊URL（如下所示的前三页URL）即可，其他操作不变
  3. 示例网站：https://www.zhipin.com/
    • 第一页网址：https://www.zhipin.com/c101270100/?query=%E6%B8%97%E9%80%8F%E6%B5%8B%E8%AF%95&page=1&ka=page-1
    • 第二页网址：https://www.zhipin.com/c101270100/?query=%E6%B8%97%E9%80%8F%E6%B5%8B%E8%AF%95&page=2&ka=page-2
    • 第三页网址：https://www.zhipin.com/c101270100/?query=%E6%B8%97%E9%80%8F%E6%B5%8B%E8%AF%95&page=3&ka=page-3
  4. 设置前三页网址：https://www.zhipin.com/c101270100/?query=%E6%B8%97%E9%80%8F%E6%B5%8B%E8%AF%95&page=[1-3]&ka=page-[1-3]
    • 说明：将有规律的参数进行处理，设置为[a-b:c]
    • [a-b:c]：a代表起始数字，b代表结束数字，c代表步长（注意：[a-b]表示取从a~b的数，步长为1）
    • [1-5:2]：获取的数值依次为1、3、5

按钮翻页

  1. 说明：当网页URL没有变化时，可用翻页按钮来遍历页面

  2. 示例网址：https://www.appinn.com/

  3. 过程：新建项目 >> 创建翻页按钮选区、创建其他选区 >> 其他操作

  4. 注意事项：必须在项目根目录_root下，先创建翻页按钮选区，再创建其他选区，并且此项目中的所有Parent Selectors的所有属性都要选择上

  5. 在项目根目录下创建大选区

  6. 进入all大选区，再选择其他小选区（小选区中的Parent Selectors属性也必须选择所有）

更多按钮

爬取二级页面的内容

  1. 创建项目 >> 获取一级页面中指定选区的所有链接选区

  2. 进入二级页面，创建自己需要的选区（可在二级页面先创建一个大选区，再创建小选区），再获取里面的详细数据

  3. 重要操作：若二级页面在其他窗口打开，只需将二级页面地址复制过来运行即可

  4. 打开二级页面，新建选区即可

技术总结

  1. 此插件通用使用步骤：创建项目 >> 创建选区 >> 数据爬去 >> 数据保存
  2. 此插件中，所有输入框内容必须大于等于三个字符
  3. 爬取的内容不同，选区中的Type属性值不同
  4. 同时获取多种数据，并且多种数据一一对应，相互联系，这时必须要先创建一个大选区（包含所需的所有数据），然后在此大选区中创建多个并列的小选区，每个小选区包含一种数据，这样可以避免各个数据之间错行，数据更加完整、正确
  5. 若大选区无法用鼠标选中，可先选择一个小选区（子选区），再一次次的按P键，依次获取相对的父选区，最终可获得所需的大选区
  6. 爬去多个页面中的内容，可利用元素选择器完成
  7. 翻页按钮和动态加载按钮区别很大；检查元素中，若按钮处没有链接，说明此按钮为动态加载按钮，反之亦然
