扫描工具

nmap扫描器

命令 描述
-sP ping 扫描
-p 指定端口范围
-sV 服务版本探测
-O（大写） 启用操作系统探测
-A 全面扫描
-oN 扫描结果保存为 txt
-v 显示扫描进度
  • 示例：
    1. nmap -sP 10.1.1.1/24 ：扫描网段在线主机 */24为网段参数，即扫描10.1.1.X网段
    2. nmap -p 21,23,3389 10.1.1.1： 扫描主机开放的指定端口（1-65535全端口扫描）加上 -sV 可以查看服务版本，
    3. namp -A 10.0.0.1 -O -oN D:\result.txt： 全面扫描，-O查看系统版本，-oN保存扫描结果到指定位置

hydra爆破工具（九头蛇）

  1. 参数：
    • -vV：显示执行过程
    • -L：后面添加账户字典
    • -P：后面添加密码字典
    • -l：后面添加账户
    • -p：后面添加密码
  2. 格式：
    • hydra IP -l 用户名 -p 密码 服务 （指定账户密码爆破）
    • hydra IP -L 用户名字典 -P 密码字典 服务 （指定账户密码字典进行爆破）
  3. 示例：
    • hydra 10.0.0.1 -L user.txt -P pass.txt telnet -vV （爆破telnet服务，并显示过程）
    • hydra 10.0.0.1 -L user.txt -P pass.txt rdp （爆破windows远程连接）
    • hydra 10.0.0.1 -L user.txt -P pass.txt ftp （爆破文件传输）
    • hydra 10.0.0.1 -L user.txt -P pass.txt smb （爆破文件共享）

  3. pwdump工具爆破：

    • 原理：系统的账户密码以hash值的形式存储于指定的文件SAM中，可用此工具获取hash值，进行撞库爆破

    • Windows账户密码存储位置（绝对路径）：C:\windows\system32\config\SAM （SAM文件需用指定的文件才能打开，如pwdump等工具）

    • 获取hash密码步骤：

      • 第一步：使用管理员权限打开cmd窗口

      • 第二步：打开DOS窗口，切换路径到工具存放的路径下，并打开工具

      • 第三步：进入工具内部，输入命令QuarksPwDump.exe -dhl，抓取hash值

  4. SAMInside工具撞库破解hash

    • 第一步：将获取到的hash值，保存在指定文件中，如：result.txt

    • 第二步：打开SAMInside工具的根目录，设置密码字典

    • 第三步：打开工具，选择hash文件，开始撞库爆破
